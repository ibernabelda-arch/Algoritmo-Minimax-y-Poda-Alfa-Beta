# Laboratorio: Algoritmo Minimax y Poda Alfa-Beta

## 1. Descripción
Este laboratorio implementa y compara el rendimiento del algoritmo clásico de búsqueda adversarial **Minimax** frente a su optimización mediante **Poda Alfa-Beta** en árboles de juego simplificados.

Desarrollado bajo el entorno de **Python 3.14** e integrado con herramientas de aseguramiento de calidad (QA).
